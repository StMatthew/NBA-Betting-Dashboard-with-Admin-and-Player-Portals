# NBA-Betting-Dashboard-with-Admin-and-Player-Portals
The admin can create games, generate unique player links, track bets, monitor the total pot, update live scores, and settle results. Players can access their own dashboard, place bets, view live match status, and check their final payout. I also designed a modern dark UI and implemented rules such as a minimum bet amount and automatic fee
